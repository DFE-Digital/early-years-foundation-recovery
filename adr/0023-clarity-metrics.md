# Use Clarity for tracking user journeys

* Status: accepted, supersedes [ADR-0018](0018-user-tracking.md)

## Context and Problem Statement

Hotjar has been decommissioned and deprecated by DfE. What alternative should we use?

## Decision Drivers

* Recommendations from DfE, and other affected projects
* Cost
* Security

## Considered Options

* Microsoft Clarity

## Decision Outcome

Use Microsoft Clarity, easy to set up and maintain, data masking and heatmap options are available
